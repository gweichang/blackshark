# 黑鲨词库
## 医学名词翻译工具

把在实验和写文章过程中积累的一些医学名词进行汇总，制作了一个查询工具，用于解决行业垂直翻译不准确的问题

当词库无法覆盖查询内容时，会自动连接到百度垂直领域翻译平台获取结果

## 搭建指南
在本地搭建此项目时，请自行注册百度垂直领域翻译平台，并修改以下配置项
```
app.id=xxxx
security.key=xxxxx
```
